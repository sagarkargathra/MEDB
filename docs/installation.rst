Installation
============

This repository is not packaged yet. The module is imported directly
from the ``src`` directory.

Requirements
------------

Python 3.10+

Required packages:

- pandas
- requests

Optional (documentation):

- sphinx

Installation example:

.. code-block:: bash

   pip install pandas requests sphinx

Repository Structure
--------------------

Expected folders:

- ``src/`` : Python module
- ``data/`` : SQLite archive
- ``scripts/`` : execution entrypoints
- ``notebooks/`` : exploratory notebooks
